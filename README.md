# NetworkingKeyTerminology
| **Protocol**                        | **Acronym**                           | ***Description***                   |
| :---------------------------------- | :----------------------------------- | :----------------------------------- |
| Wired Equivalent Privacy (*Проводной эквивалент конфиденциальности*) |  ***`WEP`*** | WEP — это тип протокола безопасности, который обычно использовался для защиты беспроводных сетей |
| Secure Shell (*Безопасная оболочка*) | ***`SSH`*** | Безопасный сетевой протокол, используемый для входа в систему и выполнения команд в удаленной системе |
| File Transfer Protocol (*Протокол передачи файлов*) | ***`FTP`*** | Сетевой протокол, используемый для передачи файлов из одной системы в другую |
| Simple Mail Transfer Protocol (*Простой протокол передачи почты*) | ***`SMTP`*** | Протокол, используемый для отправки и получения электронных писем |
| Hypertext Transfer Protocol (*Протокол передачи гипертекста*) | ***`HTTP`*** | Клиент-серверный протокол, используемый для отправки и получения данных через Интернет |
| Server Message Block (*Блок сообщений сервера*) | ***`SMB`*** | Протокол, используемый для обмена файлами, принтерами и другими ресурсами в сети |
| Network File System (*Сетевая файловая система*)| ***`NFS`*** | Протокол, используемый для доступа к файлам по сети |
| Simple Network Management Protocol (*Простой протокол управления сетью*) | ***`SNMP`*** | Протокол, используемый для управления сетевыми устройствами |
| Wi-Fi Protected Access (*Защищенный доступ к Wi-F*) | ***`WPA`*** | WPA — это протокол безопасности беспроводной сети, который использует пароль для защиты беспроводных сетей от несанкционированного доступа |
| Temporal Key Integrity Protocol (*Протокол целостности временного ключа*) | ***`TKIP`*** | TKIP также является протоколом безопасности, используемым в беспроводных сетях, но менее безопасным. |
| Network Time Protocol (*Протокол сетевого времени*) | ***`NTP`*** | Он используется для синхронизации синхронизации компьютеров в сети. |
| Virtual Local Area Network (*Виртуальная локальная сеть*) | ***`VLAN`*** | Это способ сегментации сети на несколько логических сетей |
| VLAN Trunking Protocol (*Протокол транкинга VLAN*) | ***`VTP`*** | VTP — это протокол уровня 2, который используется для создания и обслуживания виртуальной локальной сети (VLAN), охватывающей несколько коммутаторов |
| Routing Information Protocol (*Протокол информации о маршрутизации*) | ***`RIP`*** | RIP — это протокол маршрутизации вектора расстояний, используемый в локальных сетях (LAN) и глобальных сетях (WAN) |
| Open Shortest Path First (*Протокол динамической маршрутизации*) | ***`OSPF`*** | Это протокол внутреннего шлюза (IGP) для маршрутизации трафика в рамках одной автономной системы (AS) в сети интернет-протокола (IP) |
| Interior Gateway Routing Protocol (*Протокол маршрутизации внутреннего шлюза*) | ***`IGRP`*** | IGRP — это проприетарный протокол внутреннего шлюза Cisco, предназначенный для маршрутизации внутри автономных систем |
| Enhanced Interior Gateway Routing Protocol (*Усовершенствованный протокол маршрутизации внутренних шлюзов*) | ***`EIGRP`*** | Это усовершенствованный протокол векторной маршрутизации расстояний, который используется для маршрутизации IP-трафика в сети |
| Pretty Good Privacy (*Шифрования и цифровой подписи сообщений*) | ***`PGP`*** | PGP — это программа шифрования, которая используется для защиты электронной почты, файлов и других типов данных |
| Network News Transfer Protocol (*Протокол передачи новостей по сети*) | ***`NNTP`*** | NNTP - это протокол, используемый для распространения и получения сообщений в группах новостей через Интернет |
| Cisco Discovery Protocol (*Протокол Cisco Discovery*) | ***`CDP`*** | Это проприетарный протокол, разработанный Cisco Systems, который позволяет сетевым администраторам обнаруживать устройства Cisco, подключенные к сети, и управлять ими |
| Hot Standby Router Protocol (*Протокол маршрутизатора горячего резервирования*) | ***`HSRP`*** | HSRP — это протокол, используемый в маршрутизаторах Cisco для обеспечения резервирования в случае сбоя маршрутизатора или другого сетевого устройства |
| Virtual Router Redundancy Protocol (*Протокол резервирования виртуального маршрутизатора*) | ***`VRRP`*** | Это протокол, используемый для автоматического назначения доступных маршрутизаторов интернет-протокола (IP) участвующим хостам |
| Spanning Tree Protocol (*Протокол связующего дерева*) | ***`STP`*** | STP — это сетевой протокол, используемый для обеспечения топологии без петли в сетях Ethernet уровня 2 |
| Terminal Access Controller Access-Control System (*Система контроля доступа контроллера доступа терминала*) | ***`TACACS`*** | TACACS — это протокол, обеспечивающий централизованную аутентификацию, авторизацию и учет доступа к сети |
| Session Initiation Protocol (*Протокол инициации сеанса*) | ***`SIP`*** | Это сигнальный протокол, используемый для установления и завершения голосовых, видео- и мультимедийных сеансов в реальном времени по IP-сети |
| Voice Over IP (*Передача голоса по IP*) | ***`VOIP`*** | VOIP - это технология, которая позволяет совершать телефонные звонки через Интернет |
| Extensible Authentication Protocol (*Расширяемый протокол проверки подлинности*) | ***`EAP`*** | EAP — это платформа для проверки подлинности, которая поддерживает несколько методов проверки подлинности, таких как пароли, цифровые сертификаты, одноразовые пароли и проверка подлинности с открытым ключом |
| Lightweight Extensible Authentication Protocol (*Облегченный расширяемый протокол проверки подлинности*) | ***`LEAP`*** | LEAP — это проприетарный протокол беспроводной аутентификации, разработанный Cisco Systems. Он основан на протоколе EAP (Extensible Authentication Protocol), используемом в протоколе PPP (Point-to-Point Protocol) |
| Protected Extensible Authentication Protocol (*Защищенный расширяемый протокол проверки подлинности*) | ***`PEAP`*** | PEAP — это протокол безопасности, который обеспечивает зашифрованный туннель для беспроводных сетей и других типов сетей |
